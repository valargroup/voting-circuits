# ZKPs (Shielded voting)

Zero-knowledge proofs used in the shielded voting protocol:

- **[ZKP 1 — Delegation proof](zkps/zkp1-delegation-proof.md)** — Delegation of voting authority.
- **[ZKP 2 — Vote proof](zkps/zkp2-vote-proof.md)** — Casting a vote.
- **[ZKP 3 — Vote reveal proof](zkps/zkp3-vote-reveal-proof.md)** — Revealing vote choice for tally.

Content is synced from the written spec (see repo `.cursor/rules/sync-zkp.mdc` for sources).
